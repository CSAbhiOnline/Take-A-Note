**Take A Note** is a minimalist Android application built for speed and simplicity. It allows users to quickly jot down thoughts, ideas, and reminders. Powered by Google Login and Cloud Firestore, your notes are securely backed up and accessible across devices.

---

## ‚ú® Features

| Feature | Description |
| :--- | :--- |
| **Simple & Clean UI** | A clutter-free interface focused entirely on your content. No distractions. |
| **Google Login** | Secure and one-tap authentication using your existing Google account. |
| **Cloud Backup** | Never lose a thought. All notes are synced in real-time to Cloud Firestore. |
| **External Sharing** | Use the Android Intent system to share text from browsers or other apps directly into *Take A Note*. |
| **Efficient** | Lightweight architecture ensures fast startup times and smooth performance. |
